## Welcome to Huevee!
Huevee is a web app that lets you share your palette ideas. 
Build with React, Express and PostgreSQL.
Deployed in Vercel and Render.
All of the services are using free-tier resources.

### What Huevee can do?
- CRUD (Create, Read, Update, and Delete) operations.
- Every regular user can only manage their own palettes.
- The root user can delete any palette, reset reg user pass, and delete reg user.

### How to run this app?
1. Make sure you have accounts for Vercel and Render.
2. Create new PostgreSQL database in Render
